# Organizer
Aplicativo do tipo lista de tarefas feito durante o Bootcamp Mobile do Santander em parceria com a DIO

### Tecnologias e Ferramentas utilizadas:
- Android Studio
- Kotlin
- Lottie Files API para animações
- Fontes do Google: rajdhani
- Recyclerview
- Cardview
- Material Design
